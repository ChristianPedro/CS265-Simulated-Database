# CS265-Simulated-Database
The goal of this assignment was to simulate a database with bash scripting.


# Testing

## Seting up database file

```
setdb <filename>

```


## Adding to the database

```
add <object> <quantity>
```

## Deleting from database

```
delete <object>
```


## Printing contents of database

```
printdb
```

## Quiting program

```
quit
```
